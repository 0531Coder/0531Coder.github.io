html学习笔记

发表于 2017-06-14   |   分类于 前端   |     |   阅读次数

一、介绍：

  1. 浏览器访问服务器的过程：浏览器接受用户操作–>浏览器封装HTTP请求–>连接服务器：DNS解析–>发送请求Request–>服务器接受请求–>处理请求–>返回响应报文–>浏览器接收响应报文–>渲染页面呈现。
  2. DNS：Domain Name System：域名解析系统。先访问DNS服务器来获取域名对应的ip，然后再根据ip来访问服务器。
  3. 后缀名的问题：htm是为了兼容DOS系统，开发过程中后缀名使用htm或者html都可以。

    二、标签：

  4. doctype：声明html的版本。
  5. head标签是给浏览器读取的一些设置：
    • meta标签是告诉浏览器当前的编码格式。当浏览器读取的编码格式和声明的编码格式不一致的时候，会出现乱码。
    • title：设置浏览器标题栏中的显示内容。
    • Link：引入CSS样式、DNS预解析（缓存网站的ip地址）、引入网站icon的标签

      三、URL协议：

  6. scheme://host.domain:port/path/filename
    • scheme：定义因特网服务的类型，常见的http、https、ftp。
    • host：定义域主机（http默认的主机是www）。
    • domain：定义因特网域名,比如w3school.com.cn。
    • port：定义主机上的端口号（http默认的端口号是80）。
    • path：定义服务器上的路径（如果省略，则文档必须位于网站的根目录内）。
  7. url编码：
    • 避免网址（url地址）在全球都能够显示并访问，将不是ASCII编码的字符编码成%ASCII的形式。urlencode（编码）、urldecode（解码）。

四、相对路径和绝对路径：

  1. DOS命令常用操作：
    • cls ：清屏。
    • Tab键可以自动补齐命令。
    • cd demo ：进入Demo文件夹。
    • cd ../ ：返回上级目录。
    • cd ../../ ：返回上两级目录。
    • dir：展示当前目录中的文件和文件夹。
    • md：创建目录。
  2. 相对路径和绝对路径的区别：
    • 相对路径：
      是相对当前位置目标文件或者目录的位置。
      ./当前目录下；
      ../当前目录的父级目录下；
      ../../当前目录的父级的父级目录下。
    • 绝对路径：
      是指定盘符和位置。

      五、图片标签的使用

      <img src=”” title=”图片的标题” alt=”当图片不显示的时候，会显示alt属性文本”>
      src：图片的地址
      alt：当图片不显示的时候，会显示alt属性文本
      title：对网站seo有很大的影响，当鼠标移上去的时候，图片显示的tip提示文本内容。
  • 常见的图片格式：
    .gif：非常小、兼容性好、支持透明、支持动画、但是只能处理255种颜色。色彩太多不行。（logo）
    .jpg（JPEG）：有损压缩格式，文件体积小，便于在网络上传输、但是压缩成jpg格式以后容易失真。（广告banner图、海报、照片）
    .png：包括png-8和真色彩png。文件体积比gif更小、支持透明、但是会有兼容性的问题（ie6处理png-24会出现灰色的问题）。

六、标签：

  1. heading标签（h1-h6）
    补充：

    • DTD：文档类型定义。是html5的声明。
    • heading标签并不是为了页面样式的，而是为了网站的SEO优化，如果heading使用不好，会导致网站的主题不明确。样式在CSS中都可以调整。
    • 一般一个页面中只有一个h1标签，可以有多个h2标签。有标题就使用heading标签，没有标题就不使用heading标签（注意语义化）。如果heading标签滥用会被搜索引擎识别为作弊。
    • h1-h6标签对于当前文档的重要性一次降低。

  2. p标签（段落标签）
    特点：独占一行或多行。

  3. 单标签：

    • 水平线标签（<hr/>）、换行标签（<br/>）。
    • 在h5的标准中，单标签是不需要写”/“的，h5之前的版本是强制需要的。
    • html文档空白合并：在html文档中，文字之间的空格会进行合并，多个空格或者是换行会合并成一个空格。
    • 如果想让html文档换行的话，就得使用
      标签。如果想加多个空格就使用 ,这样html文档会识别为空格。

  4. 其他标签：

    • <span></span>：文本节标签，对某几个文本进行样式修改。
    • <em></em>：强调标签。局部文本的加强语气或者文本强调。加上以后文本会变成斜体，但是em不是用来对文本样式的修改，主要是包裹文档中的中心文本，进行着重强调，有利于网站的SEO优化。
    • <i></i>：定义文本成为斜体，纯粹对网页文档的样式进行修改。没有语义化功能，不推荐使用，样式的修改推荐使用CSS进行修改。
    • <strong></strong>：定义加重语气，对整个文档或者段落进行加重语气。没有语义化作用。注意与<em></em>的区别：整体与部分。
    • <b></b>:加粗标签，单纯样式的改变。
    • <sub></sub>：下标文字
    • <sup></sup>：上标文字
    • <del></del>：添加删除线

七、超链接和图片：

  1. <a></a>：超链接标签

    • herf：目标页面的地址
    • target：指定在哪个窗口打开目标页面（_blank，_self），_self表示在当前页签打开目标链接，_blank表示在新窗口打开目标链接。如果超链接的目标路径没有确定或者没有，就写一个#。

  2. 锚点链接：跳转到某个span区域并且该区域放在页面的最上方（如果当前页面够长）。
    <a href=’#demoplace’>跳转到当前页面的id为demoplace的span区域。
    <span id=’demoplace’>跳转到当前区域</span>。
    注意：超链接不仅仅可以嵌套文本，还可以嵌套表格和图片。

八、高级标签

  1. 列表标签

    • 无序列表：
      <ul><li></li><ul>：ul标签之间不嵌套其他标签，但是li之间可以嵌套其他任何标签。
    • 有序列表：
      <ol><li></li></li>：ul标签之间不嵌套其他标签，但是li之间可以嵌套其他任何标签。

    • 自定义列表：<dl></dl>：
      <dt></dt>：列表的title，里面包裹文本标签（等）
      <dd></dd>：列表项

    • 列表嵌套：
      <table></table>：表格标签。
      <tr></tr>：表示行。
      <td></td>：表示列。
      <thead></thead>：表格中的表头，包裹<tr><td>标签，<th>标签包裹不同的表头,使用<tr>（行头部）或者<td>(列头部)标签包裹。
      <tbody></tbody>：表格中的单元格，就是将<tr><td>标签使用<tbody>标签包裹。

    • 列表单元格的合并：
      如果跨行合并就在<tr>标签中使用rowspan=num，如果跨列合并就在<td>标签中使用colspan=num。

    • 节标签：<div>和<span>标签。
      <div>：块状标签，独占一行。
      <span>：行内标签，不会独占一行，与行内的其他标签共占一行。
      <p>、<div>、<ul>、<ol>、<dl>、<li>、<dt>、h1~h6：这些标签都是块级标签。
      <span><strong><em>等：文本相关的标签都是行内标签。

    • 内联框架标签（不建议使用，使用div替代）：

      iframe标签（这是一个行内标签），能够在当前html页面中完全显示目标地址的页面。

    • meta标签：可是设置关键字（SEO优化）。
    • link标签：引入css
    • base标签：可以让当前页面的所有的a标签都有相同的属性。
    • pre标签：保留原来的文本格式（空格、换行）

九、表单：

  • 在html中一个完整的表单通常由表单控件、提示信息、表单域三个部分构成。
  • 组合表单的标题：formset lenged是定义表单的头部。
  • radio标签可以使用label标签来实现点击文字选中某个选项
  • label标签可以帮助某个控件获取焦点。
  • select标签：可以表示下拉列表和普通的列表。option可以定义select的选项。
  • textarea标签：输入大量文本的时候使用。可以设置行数rows和列数cols。

十、字符实体：

  • 实现在html中页面中显示html的代码。相当于java中的转义字符。
  • 常用：©：版权符号，®：注册商标

html总结： 

1. html：页面结构。    
 2. css：页面的样式。    
3. JavaScript：页面的交互效果。
